# Python Files for Solder Station with new fixture
Solder Station based on Raspberry Pi platform which works with MEGA I/O, which provides following additional features compared to the existing setup : 
- Process Enforcement
- Test Data Logging
- User Interface inline with other test stations, Component Traceability, Battery Voltage measurement and Reverse battery Detection and pneumatic actuation of clamps to hold the boards while soldering.
